#  RavenChat
>A self-hosted cross-border chat system built with FastAPI and PWA, deployed on a TV box for 24/7 operation.

##  Project Overview
RavenChat is a fully functional self-hosted chat system I built, supporting real-time messaging, file uploads, and cross-network access via reverse proxy and intranet penetration. The frontend is built as a Progressive Web App (PWA), enabling users to access the service like a native app.

##  Tech Stack
- **Backend**: Python, FastAPI
- **Frontend**: HTML/CSS/JavaScript, PWA
- **Network**: Reverse Proxy, Intranet Penetration
- **Data Persistence**: JSON file storage
- **Other**: Custom icon, static resource management

##  Core Features
-  Real-time text chat
-  Image/file upload support
-  Chat history persistence
-  PWA-style access (desktop shortcut)
-  Cross-network access via reverse proxy
